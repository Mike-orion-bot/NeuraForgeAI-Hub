# 🔱 NeuraForgeAI-Hub
Ecosistema modular de administración y monetización de bots financieros.

## Módulos Principales
- **Core/Monetization**: Módulos de firma dinámica, SDK de licencias y micro-tasas.
- **Apps**: Panel de administración y nodo operativo.
- **Integraciones**: OrionCash (gestión de pagos) y motor antifraude.
